# Dessert Recipe Generator
> Generate cosmic dessert recipes or search for your favorites. Perfect for sweet inspiration across the galaxy.

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen)](https://vcookie-dessert-recipe-generator.netlify.app)
[![GitHub Repo](https://img.shields.io/badge/GitHub-Repo-blue)](https://github.com/Vengefulcookie/dessert-recipe-generator)

## What This Project Does
When you're craving something sweet but don't know what — this app finds a recipe for you. Click "Generate" and get a random dessert, or search for something specific like "chocolate cake" or "cheesecake." Each recipe comes with ingredients and instructions.

**What you can do:**
- Generate random dessert recipes
- Search for specific desserts by name
- View ingredients and step-by-step instructions
- Get inspired for your next baking adventure

## How I Built This
- **JavaScript** - Core application logic
- **Recipe API (Edamam/TheMealDB)** - For recipe data
- **CSS Animations** - For the fun, playful interface

## Try It Yourself
[Click here to open the live app](https://vcookie-dessert-recipe-generator.netlify.app)

## What I Learned
This project was all about working with API data and presenting it in a user-friendly way. I learned how to handle different API response structures, create fallbacks for missing data, and design an interface that feels delightful to use. The random generation feature taught me about managing state when data changes unexpectedly.
